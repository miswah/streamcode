Filter Even Numbers from a List

Problem

Given a list of integers numbers, return a new list that contains only the even numbers from the original list, preserving their original order. You must implement the solution using the Java Stream API.

Function Signature
public List<Integer> filterEvenNumbers(List<Integer> numbers)
Constraints
  • 1 <= numbers.size() <= 10^5
  • -10^9 <= numbers[i] <= 10^9
  • Order of numbers must be preserved.
Expected Time Complexity

O(n) — where n is the length of the input list.

Hint

Use numbers.stream(), filter(), and collect(Collectors.toList()).

Examples
Example 1 
Input:
numbers = [1, 2, 3, 4, 5, 6]

Output:
[2, 4, 6]

Explanation:
All even numbers retained in original order.
Example 2 
Input:
numbers = [7, 9, 13, 15]

Output:
[]

Explanation:
No even numbers present.
Example 3 
Input:
numbers = [10, 21, 32, 43, 54]

Output:
[10, 32, 54]
Edge Example 
Input:
numbers = []

Output:
[]

Explanation:
Empty input list returns empty list.
Solution 
import java.util.*;
import java.util.stream.Collectors;

public class Solution {
    public List<Integer> filterEvenNumbers(List<Integer> numbers) {
        // Implement using Java Stream API
        return numbers.stream()
                      .filter(n -> n % 2 == 0)
                      .collect(Collectors.toList());
    }

    public static void main(String[] args) {
        Solution sol = new Solution();
        List<Integer> nums = Arrays.asList(1,2,3,4,5,6);
        System.out.println(sol.filterEvenNumbers(nums)); // [2, 4, 6]
    }
}
