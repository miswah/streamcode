Find Maximum in a List

Problem

Given a list of integers numbers, return the maximum number in the list. You must implement the solution using the Java Stream API.

Function Signature
public int findMaximum(List<Integer> numbers)
Constraints
  • 1 <= numbers.size() <= 10^5
  • -10^9 <= numbers[i] <= 10^9
Expected Time Complexity

O(n) — where n is the length of the input list.

Hint

Use numbers.stream().max(Comparator.naturalOrder()).

Examples
Example 1 
Input:
numbers = [3, 1, 4, 1, 5, 9]

Output:
9

Explanation:
The maximum number in the list is 9.
Example 2 
Input:
numbers = [-5, -2, -8, -1]

Output:
-1

Explanation:
The maximum number is -1, even though all numbers are negative.
Edge Example 
Input:
numbers = [7]

Output:
7

Explanation:
Single element list returns that element as maximum.
Solution 
import java.util.*;
import java.util.stream.Collectors;

public class Solution {
    public int findMaximum(List<Integer> numbers) {
        // Implement using Java Stream API
        return numbers.stream()
                      .max(Comparator.naturalOrder())
                      .orElseThrow(() -> new NoSuchElementException("List is empty"));
    }

    public static void main(String[] args) {
        Solution sol = new Solution();
        List<Integer> nums = Arrays.asList(3,1,4,1,5,9);
        System.out.println(sol.findMaximum(nums)); // 9
    }
}
